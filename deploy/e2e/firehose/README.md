# firehose

Synthetic logs/metrics/traces load generator, vendored (copied, not a
submodule) from `terraform-google-monitoring`'s
`traffic-gen/loaders/firehose` for use by
[`deploy/e2e/compose.grafana.yaml`](../compose.grafana.yaml)'s local
Grafana demo stack. See that file's header comment for how to run it.

Source of truth is the origin repo — changes here are a point-in-time
copy and won't sync automatically.
